## 🐛 Troubleshooting

**Port 5173 in use?**
```bash
npm run dev -- --port 3000
```

**Module not found?**
```bash
npm install
```

**Need clean slate?**
```bash
npm install
rm -rf node_modules
npm cache clean --force
npm install
```

**Want to reset data?**
Open console (F12): `localStorage.clear()`
